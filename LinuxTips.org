** [[https://github.com/archlinuxcn/repo][Archlinuxcn]] source
1. edit /etc/pacman.conf, add
#+begin_src conf
  [archlinux]
  Server = https://repo.archlinuxcn.org/$arch
#+end_src
2. import PGP Keys:
#+begin_src sh
  sudo pacman -Sy && sudo pacman -S archlinxucn-keyring
#+end_src
